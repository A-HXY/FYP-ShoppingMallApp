Project Description

This project outlines the development of a mobile e-commerce platform, the Easy To Go Shopping Mall App. The app is designed to provide a user-friendly shopping experience, allowing users to browse products, add items to a cart, make secure payments, and manage orders.

Key Technical Highlights

  • Agile development methodology for iterative progress.
  • Kotlin-based Android front end for smooth user experience.
  • Spring Boot backend integrated with MySQL database.
  • Scalability, security, and seamless third-party integration (e.g., payment gateways).
